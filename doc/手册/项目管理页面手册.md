# 项目管理页面手册

## 概述

项目管理页面用于管理系统中的项目配置，支持创建、编辑、删除和搜索项目，以及关联工作区和模型。

## 功能说明

### 1. 项目列表

| 字段 | 说明 | 数据来源 |
|------|------|----------|
| 项目名称 | 项目显示名称 | `project.name` |
| 描述 | 项目描述 | `project.description` |
| 状态 | 活跃/停用 | `project.status` |
| 会话数 | 项目下的会话数量 | `project.session_count` |
| 工作区 | 关联的工作区名称 | `project.workspace_config_id` |
| 模型 | 关联的模型名称 | `project.model_config_id` |
| 创建时间 | 创建日期 | `project.created_at` |

### 2. 项目状态

| 状态值 | 显示名称 | 说明 |
|--------|----------|------|
| active | 活跃 | 项目可正常使用 |
| inactive | 停用 | 项目不可用 |

### 3. 搜索功能

- 实时搜索项目名称
- 搜索范围：项目名称

### 4. 新建/编辑项目

| 表单字段 | 类型 | 必填 | 默认值 |
|----------|------|------|--------|
| 项目名称 | 文本输入 | 是 | 空 |
| 描述 | 文本域 | 否 | 空 |
| 关联工作区 | 下拉选择 | 否 | 空 |
| 关联模型 | 下拉选择 | 否 | 空 |
| 状态 | 下拉选择 | 否 | active |

## 事件监听

| 事件 | 触发时机 |
|------|----------|
| `project:loaded` | 项目数据加载完成 |
| `project:created` | 新项目创建成功 |
| `project:updated` | 项目更新成功 |
| `project:deleted` | 项目删除成功 |

## 核心方法

| 方法 | 功能 |
|------|------|
| `loadProjects()` | 加载项目列表 |
| `loadWorkspaces()` | 加载工作区列表（用于下拉选择） |
| `loadModels()` | 加载模型列表（用于下拉选择） |
| `handleSearch()` | 处理搜索 |
| `showCreateModal()` | 显示新建弹窗 |
| `handleEdit(project)` | 显示编辑弹窗 |
| `handleSave()` | 保存项目 |
| `handleDelete(id)` | 删除项目 |
| `getWorkspaceName(id)` | 获取工作区名称 |
| `getModelName(id)` | 获取模型名称 |

## 使用流程

1. **搜索项目**: 在搜索框输入关键词
2. **新建项目**: 点击「+ 新建项目」按钮
3. **填写表单**: 输入项目名称、描述、选择关联的工作区和模型
4. **保存**: 点击保存按钮
5. **编辑/删除**: 在列表中操作

## 注意事项

1. 项目可以关联工作区和模型
2. 工作区和模型需要提前创建
3. 删除项目会同时删除关联的会话
4. 删除项目需要确认操作
