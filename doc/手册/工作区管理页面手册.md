# 工作区管理页面手册

## 概述

工作区管理页面用于管理代码工作区配置，包括工作区的创建、编辑、删除和搜索。

## 页面结构

```
┌─────────────────────────────────────────────────────────────┐
│  📂 工作区管理                                    + 新建工作区 │
├─────────────────────────────────────────────────────────────┤
│  [搜索框: 搜索工作区...]                                      │
├─────────────────────────────────────────────────────────────┤
│  ┌──────────┬──────────┬──────┬──────┬──────┬──────┐        │
│  │工作区名称│根路径    │类型  │编码  │创建时间│操作  │        │
│  ├──────────┼──────────┼──────┼──────┼──────┼──────┤        │
│  │ Workspace│ /path/to │本地  │utf-8 │...    │编辑  │        │
│  │          │ /project │      │      │       │删除  │        │
│  └──────────┴──────────┴──────┴──────┴──────┴──────┘        │
```

## 功能说明

### 1. 工作区列表

| 字段 | 说明 | 数据来源 |
|------|------|----------|
| 工作区名称 | 工作区显示名称 | `workspace.name` |
| 根路径 | 工作区根目录路径 | `workspace.root_path` |
| 类型 | 本地或远程 | `workspace.type` |
| 编码 | 文件编码格式 | `workspace.encoding` |
| 创建时间 | 创建日期 | `workspace.created_at` |

### 2. 工作区类型

| 类型值 | 显示名称 | 说明 |
|--------|----------|------|
| local | 本地 | 本地文件系统工作区 |
| remote | 远程 | 远程服务器工作区 |

### 3. 搜索功能

- 实时搜索工作区名称
- 搜索范围：工作区名称

### 4. 新建/编辑工作区

| 表单字段 | 类型 | 必填 | 默认值 |
|----------|------|------|--------|
| 工作区名称 | 文本输入 | 是 | 空 |
| 根路径 | 文本输入 | 否 | 空 |
| 类型 | 下拉选择 | 否 | local |
| 编码 | 文本输入 | 否 | utf-8 |

## 数据结构

### 工作区对象

```json
{
  "id": "workspace_001",
  "name": "我的工作区",
  "root_path": "/path/to/project",
  "type": "local",
  "encoding": "utf-8",
  "excluded_patterns": ["node_modules", ".git"],
  "created_at": "2024-01-01T00:00:00Z"
}
```

## 事件监听

| 事件 | 触发时机 |
|------|----------|
| `workspace:loaded` | 工作区数据加载完成 |
| `workspace:created` | 新工作区创建成功 |
| `workspace:updated` | 工作区更新成功 |
| `workspace:deleted` | 工作区删除成功 |

## 核心方法

| 方法 | 功能 |
|------|------|
| `loadWorkspaces()` | 加载工作区列表 |
| `handleSearch()` | 处理搜索 |
| `showCreateModal()` | 显示新建弹窗 |
| `handleEdit(workspace)` | 显示编辑弹窗 |
| `handleSave()` | 保存工作区 |
| `handleDelete(id)` | 删除工作区 |

## 使用流程

1. **搜索工作区**: 在搜索框输入关键词
2. **新建工作区**: 点击「+ 新建工作区」按钮
3. **填写表单**: 输入工作区名称、根路径、选择类型和编码
4. **保存**: 点击保存按钮
5. **编辑/删除**: 在列表中操作

## 注意事项

1. 工作区根路径应使用绝对路径
2. 编码默认为 utf-8
3. 排除模式用于忽略特定文件或目录
4. 删除工作区需要确认操作
