# 对话页面手册

## 概述

对话页面是 Trae AI 助手的核心交互界面，用户通过此页面与 AI 助手进行对话交流。

## 页面结构

```
┌─────────────────────────────────────────────────────────────────┐
│                         导航栏                                 │
├─────────────────────────────────────────────────────────────────┤
│  ┌─────────────────┐  ┌─────────────────────────────────────┐  │
│  │   会话侧边栏     │  │          对话主区域                  │  │
│  │  ┌───────────┐  │  │  ┌─────────────────────────────┐   │  │
│  │  │ 会话列表   │  │  │  │     消息展示区域             │   │  │
│  │  │           │  │  │  │  [用户消息]                  │   │  │
│  │  │ + 新建会话 │  │  │  │  [助手响应]                  │   │  │
│  │  └───────────┘  │  │  │  [工具调用卡片]              │   │  │
│  │                 │  │  └─────────────────────────────┘   │  │

│                       │  │  [文本输入框] [发送按钮]       │   │  │
│                       │  └─────────────────────────────┘   │  │
│                       └─────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────────┘
```

## 功能说明

### 1. 会话侧边栏

| 功能 | 说明 |
|------|------|
| 会话列表 | 显示当前项目下的所有会话 |
| 新建会话 | 创建新的对话会话 |
| 项目选择器 | 切换不同项目 |

### 2. 对话主区域

| 组件 | 说明 |
|------|------|
| 消息展示区 | 显示用户和助手的消息历史 |
| 输入区域 | 用户输入消息的文本框 |

### 3. 消息类型

| 类型 | 说明 | 组件 |
|------|------|------|
| 用户消息 | 用户发送的消息 | UserMessageComponent |
| 助手消息 | AI助手的响应 | AssistantMessageComponent |
| 思考块 | AI的思考过程 | ThinkBlockComponent |
| 工具调用 | 工具调用记录 | ToolCardComponent |
| 文本块 | 格式化文本内容 | TextBlockComponent |

## 组件结构

### 核心组件

| 组件 | 路径 | 功能 |
|------|------|------|
| ChatPageComponent | `components/chat/ChatPageComponent.js` | 主页面组件 |
| ChatSidebarComponent | `components/chat/ChatSidebarComponent.js` | 侧边栏 |
| UserMessageComponent | `components/chat/UserMessageComponent.js` | 用户消息 |
| AssistantMessageComponent | `components/chat/AssistantMessageComponent.js` | 助手消息 |
| ChatInputComponent | `components/chat/ChatInputComponent.js` | 输入组件 |

### 服务依赖

| 服务 | 功能 |
|------|------|
| ChatManager | 管理对话状态和消息流 |
| StateManager | 管理全局状态 |
| EventBus | 事件总线 |

## 事件监听

| 事件 | 触发时机 | 处理逻辑 |
|------|----------|----------|
| `session:loaded` | 会话数据加载完成 | 更新会话列表 |
| `session:switching` | 切换会话时 | 清空消息，重置状态 |
| `state:updated` | 状态更新时 | 更新消息列表 |

## 数据流

```
用户输入 → ChatInputComponent → ChatManager → 后端API → 
→ LLM响应 → ChatManager → EventBus → ChatPageComponent → 消息展示
```

## 使用流程

1. **选择项目**: 通过项目选择器选择目标项目
2. **选择会话**: 从侧边栏选择或创建会话
3. **发送消息**: 在输入框中输入消息并发送
4. **查看响应**: 助手响应会实时显示在消息区域

## 状态管理

| 状态键 | 说明 |
|--------|------|
| `sessions` | 当前项目的会话列表 |
| `currentProjectId` | 当前项目ID |
| `currentSessionId` | 当前会话ID |
| `messages` | 当前会话的消息列表 |

## 注意事项

1. 页面使用 Vue.js 组件化开发
2. 消息通过 EventBus 进行实时更新
3. 状态通过 StateManager 进行持久化
4. 会话切换时会清空当前消息并重置状态
