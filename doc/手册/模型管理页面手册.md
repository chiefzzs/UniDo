# 模型管理页面手册

## 概述

模型管理页面用于管理AI模型配置，支持创建、编辑、删除和搜索模型配置。

├─────────────────────────────────────────────────────────────┤

## 功能说明

### 1. 模型列表

| 字段 | 说明 | 数据来源 |
|------|------|----------|
| 模型名称 | 模型显示名称 | `model.name` |
| 模型类型 | 模型类型 | `model.type` |
| API地址 | 模型API端点 | `model.apiUrl` |
| 状态 | 启用/停用 | `model.status` |
| 创建时间 | 创建日期 | `model.created_at` |

### 2. 模型类型

| 类型值 | 显示名称 | 说明 |
|--------|----------|------|
| chat | 对话模型 | 用于对话交互 |
| embedding | 嵌入模型 | 用于文本嵌入 |
| image | 图像模型 | 用于图像生成 |

### 3. 搜索功能

- 实时搜索模型名称
- 搜索范围：模型名称

### 4. 新建/编辑模型

| 表单字段 | 类型 | 必填 | 默认值 |
|----------|------|------|--------|
| 模型名称 | 文本输入 | 是 | 空 |
| 模型类型 | 下拉选择 | 否 | chat |
| API地址 | 文本输入 | 否 | 空 |
| API密钥 | 密码输入 | 否 | 空 |

## 事件监听

| 事件 | 触发时机 |
|------|----------|
| `model:loaded` | 模型数据加载完成 |
| `model:created` | 新模型创建成功 |
| `model:updated` | 模型更新成功 |
| `model:deleted` | 模型删除成功 |

## 核心方法

| 方法 | 功能 |
|------|------|
| `loadModels()` | 加载模型列表 |
| `handleSearch()` | 处理搜索 |
| `getModelTypeName(type)` | 获取模型类型显示名称 |
| `showCreateModal()` | 显示新建弹窗 |
| `handleEdit(model)` | 显示编辑弹窗 |
| `handleSave()` | 保存模型 |
| `handleDelete(id)` | 删除模型 |

## 使用流程

1. **搜索模型**: 在搜索框输入关键词
2. **新建模型**: 点击「+ 新建模型」按钮
3. **填写表单**: 输入模型名称、选择类型、填写API地址和密钥
4. **保存**: 点击保存按钮
5. **编辑/删除**: 在列表中操作

## 注意事项

1. API密钥会被加密存储
2. 模型配置完成后需要测试连接
3. 删除模型需要确认操作
4. 模型可以被多个项目关联
