# 提示词管理页面手册

## 概述

提示词管理页面用于管理系统中的提示词配置，支持创建、编辑、删除和搜索提示词。

## 页面结构

├─────────────────────────────────────────────────────────────┤
│  ┌──────────┬──────┬──────┬──────┬──────┬──────┐            │
│  │提示词名称│分类  │变量  │状态  │创建时间│操作  │            │

## 功能说明

### 1. 提示词列表

| 字段 | 说明 | 数据来源 |
|------|------|----------|
| 提示词名称 | 提示词的显示名称 | `prompt.name` |
| 分类 | 提示词类型 | `prompt.category` |
| 变量 | 提示词中使用的变量 | `prompt.variables` |
| 状态 | 启用/停用 | `prompt.is_active` |
| 创建时间 | 创建日期 | `prompt.created_at` |

### 2. 提示词分类

| 分类值 | 显示名称 | 用途 |
|--------|----------|------|
| system_prompt | 系统提示词 | 指导AI行为的核心指令 |
| user_prompt | 用户提示词 | 用户自定义提示词 |
| assistant_prompt | 助手提示词 | 助手角色定义 |

### 3. 变量支持

提示词内容支持 `{{variable}}` 格式的变量替换。

### 4. 搜索功能

- 实时搜索提示词名称
- 搜索范围：提示词名称

### 5. 新建/编辑提示词

| 表单字段 | 类型 | 必填 |
|----------|------|------|
| 提示词名称 | 文本输入 | 是 |
| 分类 | 下拉选择 | 是 |
| 内容 | 文本域 | 是 |
| 启用状态 | 复选框 | 否 |

## 事件监听

| 事件 | 触发时机 |
|------|----------|
| `prompt:loaded` | 提示词数据加载完成 |
| `prompt:created` | 新提示词创建成功 |
| `prompt:updated` | 提示词更新成功 |
| `prompt:deleted` | 提示词删除成功 |

## 核心方法

| 方法 | 功能 |
|------|------|
| `loadPrompts()` | 加载提示词列表 |
| `handleSearch()` | 处理搜索 |
| `showCreateModal()` | 显示新建弹窗 |
| `handleEdit(prompt)` | 显示编辑弹窗 |
| `handleSave()` | 保存提示词 |
| `handleDelete(id)` | 删除提示词 |

## 使用流程

1. **搜索提示词**: 在搜索框输入关键词
2. **新建提示词**: 点击「+ 新建提示词」按钮
3. **填写表单**: 输入提示词名称、选择分类、填写内容
4. **保存**: 点击保存按钮
5. **编辑/删除**: 在列表中操作

## 注意事项

1. 提示词内容支持变量替换，格式为 `{{变量名}}`
2. 系统提示词通常在对话开始时自动注入
3. 删除提示词需要确认操作
