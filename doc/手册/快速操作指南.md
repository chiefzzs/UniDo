# 道一（UniDo）- 快速操作指南

## 快速上手

### 0）启动工具

```powershell
# 进入项目目录
cd d:\learnning\260521

# 启动服务
python run.py
```

服务启动后，访问 http://localhost:8000

---

### 1）模型配置

**配置步骤：**

1. 打开 **工具菜单** → **模型管理**
2. 点击 **+ 新建模型**
3. 填写配置信息：
   - **模型名称**：如 `GPT-4`
   - **模型类型**：对话模型
   - **API地址**：OpenAI API 地址（如 `https://api.openai.com/v1`）
   - **API密钥**：您的 OpenAI API Key
4. 点击 **保存**

---

### 2）工作区配置

**配置步骤：**

1. 打开 **工具菜单** → **工作区管理**
2. 点击 **+ 新建工作区**
3. 填写配置信息：
   - **工作区名称**：如 `我的项目`
   - **根路径**：本地项目路径（如 `d:\projects\myapp`）
   - **类型**：本地
   - **编码**：utf-8
4. 点击 **保存**

---

### 3）项目配置

**配置步骤：**

1. 打开 **工具菜单** → **项目管理**
2. 点击 **+ 新建项目**
3. 填写配置信息：
   - **项目名称**：如 `我的项目`
   - **关联工作区**：选择步骤2创建的工作区
   - **关联模型**：选择步骤1创建的模型
   - **状态**：活跃
4. 点击 **保存**

---

### 4）开始对话

**操作步骤：**

1. 在左侧项目选择器中，选择已创建的项目
2. 点击 **+** 按钮创建新会话
3. 在输入框中输入您的问题
4. 按 Enter 或点击发送按钮
5. 等待道一响应

## 功能模块

| 模块 | 功能 | 路径 |
|------|------|------|
| 对话页面 | 与道一交流 | 主界面 |
| 提示词管理 | 管理提示词模板 | 工具菜单 → 提示词管理 |
| 模型管理 | 配置 AI 模型 | 工具菜单 → 模型管理 |
| 工作区管理 | 管理代码工作区 | 工具菜单 → 工作区管理 |
| 项目管理 | 管理项目配置 | 工具菜单 → 项目管理 |
| 工具管理 | 管理工具配置 | 工具菜单 → 工具管理 |
| 存储配置 | 配置数据持久化 | 工具菜单 → 存储配置 |

---

## 注意事项

1. **API密钥安全**：不要将 API 密钥提交到公开仓库
2. **工作区路径**：建议使用绝对路径
3. **项目关联**：确保项目关联了有效的工作区和模型
4. **工具激活**：检查工具管理页面，确保需要的工具已激活

---

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| 无法连接模型 | 检查 API 地址和密钥是否正确 |
| 工作区无法访问 | 确认路径存在且有权限 |
| 工具不响应 | 检查工具是否已激活 |
| 页面空白 | 刷新页面或重启服务 |
