# 项目说明

## 📁 目录结构

```
/
├── README.md                # 项目说明
├── static_src/              # 源代码目录（开发用）
│   ├── main.js             # 应用入口
│   ├── package.json        # npm 配置
│   ├── 打包.bat            # 手动打包脚本
│   ├── 自动打包监控.bat     # 自动监控脚本
│   ├── js/
│   │   └── components/     # Vue 组件
│   │       ├── admin/      # 管理页面组件
│   │       ├── chat/       # 聊天组件
│   │       ├── infrastructure/  # 基础设施
│   │       ├── nav/        # 导航组件
│   │       └── services/   # 服务层
│   └── node_modules/       # npm 依赖
└── src/newstatic/          # 生产目录（网站文件）
    ├── index.html          # 入口 HTML
    └── js/
        ├── bundle.js       # 打包后的 JS（67KB）
        └── bundle.js.map   # Source Map
```

## 🚀 快速开始

### 1. 安装依赖（首次使用）
```bash
cd static_src
npm install
```

### 2. 启动自动监控（推荐）
```bash
# 双击运行
双击：static_src/自动打包监控.bat
```

### 3. 修改代码
编辑 `static_src/js/components/` 下的组件文件

### 4. 保存并刷新
- 保存文件（Ctrl+S）
- 等待 12 毫秒自动打包
- 刷新浏览器（Ctrl+Shift+R）

## 🛠️ 开发工具

### 手动打包
```bash
# 方式 1：使用批处理
双击：static_src/打包.bat

# 方式 2：使用 npm
cd static_src
npm run build
```

### 自动监控
```bash
# 方式 1：使用批处理
双击：static_src/自动打包监控.bat

# 方式 2：使用 npm
cd static_src
npm run build:dev
```

## 📊 性能

- **打包速度**：12 毫秒
- **文件大小**：67KB（已压缩）
- **加载时间**：1-2 秒

## 📝 注意事项

1. **开发时**：只修改 `static_src/` 目录下的文件
2. **生产时**：`src/newstatic/` 目录只放网站必要的文件
3. **首次使用**：需要在 `static_src/` 目录运行 `npm install`
