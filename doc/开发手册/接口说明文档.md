## HTTP API 接口

### 基础信息

| 属性 | 值 |
|------|------|
| 基础URL | `http://localhost:8000/api` |

### 项目管理接口

#### 创建项目

```
POST /api/projects
```

**请求体**：
```json
{
  "name": "string",           // 必填，项目名称
  "model_id": "string",       // 必填，关联模型ID
  "workspace_id": "string",   // 必填，关联工作区ID
  "description": "string"     // 可选，项目描述
}
```

**响应体**：
```json
{
  "project_id": "proj-xxx",
  "name": "AI助手开发",
  "model_id": "model-xxx",
  "workspace_id": "ws-xxx",
  "description": "道一开发项目",
  "status": "active",
  "created_at": "2026-06-05T10:00:00"
}
```

#### 获取项目列表

```
GET /api/projects
```

**查询参数**：
| 参数 | 类型 | 说明 |
|------|------|------|
| `status` | string | 过滤状态（active/inactive） |

**响应体**：
```json
{
  "projects": [
    {
      "project_id": "proj-xxx",
      "name": "AI助手开发",
      "model_id": "model-xxx",
      "workspace_id": "ws-xxx",
      "status": "active"
    }
  ]
}
```

#### 获取单个项目

```
GET /api/projects/{project_id}
```

**响应体**：
```json
{
  "project_id": "proj-xxx",
  "name": "AI助手开发",
  "model": {
    "model_id": "model-xxx",
    "name": "OpenAI GPT-4"
  },
  "workspace": {
    "workspace_id": "ws-xxx",
    "path": "D:\\projects\\my-app"
  }
}
```

#### 更新项目

```
PUT /api/projects/{project_id}
```

**请求体**：
```json
{
  "name": "string",
  "model_id": "string",
  "workspace_id": "string",
  "description": "string"
}
```

#### 删除项目

```
DELETE /api/projects/{project_id}
```

### 模型配置接口

#### 创建模型配置

```
POST /api/models
```

**请求体**：
```json
{
  "name": "string",           // 必填，模型配置名称
  "api_type": "string",       // 必填，API类型（openai/anthropic/qwen）
```

```json
  "model_name": "gpt-4",

  "temperature": 0.7,
  "status": "active",
```

```python
class PersistenceService:
```

#### LLM Client

```python
# 接口定义
class LLMClient:
    def call(request: LLMRequest) -> LLMResponse
    def call_stream(request: LLMRequest) -> Iterator[StreamChunk]
    def set_mode(mode: str) -> None  # 'record' | 'loopback'
```

**LLMRequest 数据结构**：
```python
@dataclass
class LLMRequest:
    model_name: str
    messages: List[Dict[str, str]]
    temperature: float = 0.7
    max_tokens: int = 2048
    stream: bool = False
    api_type: str = 'openai'
    api_address: str = 'http://localhost'
    api_key: str = 'demo'
    tools: List[Dict] = None
    tool_choice: str = None
```

**LLMResponse 数据结构**：
```python
@dataclass
class LLMResponse:
    content: str
    finish_reason: str
    model_name: str
    thinking: str = ""
    usage: Dict[str, int] = {}
    tool_calls: List[Dict] = []
```

**StreamChunk 数据结构**：
```python
@dataclass
class StreamChunk:
    chunk_id: str
    delta: str
    chunk_type: str  # 'thinking' | 'reasoning' | 'text' | 'tool_call'
    finish_reason: str | None = None
    index: int = 0
    tool_calls: List[Dict] = []
```

#### EventBus

```python
# 接口定义
class EventBus:
    def publish(event: Event, correlation_id: str = None) -> str
    def subscribe(event_type: str, handler: Callable) -> Subscription
    def unsubscribe(subscription: Subscription) -> None
    def get_events(correlation_id: str) -> List[EventRecord]
```

**Event 数据结构**：
```python
@dataclass
class Event:
    event_type: str
    payload: Dict[str, Any]
    source: str = ""
    timestamp: str = None
    
    def to_record(correlation_id: str, stored_at: str) -> EventRecord
```
