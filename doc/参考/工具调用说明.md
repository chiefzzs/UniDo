# 大模型 Function Call 历史消息示例文档
## 格式规范说明
- `assistant`：携带 `tool_calls` 发起工具调用
- `tool`：携带 `tool_call_id` 回传工具执行结果/失败信息
- 覆盖场景：多任务串行执行、工具失败纠错重试、嵌套任务拆解

---

## 场景1：用户输入划分多个任务，每个任务调用多个工具，逐一执行完成
```json
[
  {
    "role": "system",
    "content": "你是任务拆解Agent，将用户需求拆成多个任务，每个任务调用对应工具，串行执行，全部完成后汇总回答。"
  },
  {
    "role": "user",
    "content": "查今天北京天气，计算体感舒适度，读取本地历史天气对比"
  },
  {
    "role": "assistant",
    "tool_calls": [
      {
        "id": "call_1",
        "type": "function",
        "function": {
          "name": "search",
          "arguments": "{\"city\":\"北京\",\"type\":\"today_weather\"}"
        }
      }
    ]
  },
  {
    "role": "tool",
    "tool_call_id": "call_1",
    "content": "北京今日：晴，22-30℃，湿度40%"

  {

        "id": "call_2",
        "type": "function",
        "function": {
          "name": "calculate",
          "arguments": "{\"temp\":26,\"humidity\":40}"
        }
      }
    ]
  },
  {
    "role": "tool",
    "tool_call_id": "call_2",
    "content": "舒适度指数：75，舒适"
  },

    "role": "assistant",
```
