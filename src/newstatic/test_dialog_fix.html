对话归属修复测试工具

此工具用于验证第二次对话的工具调用是否正确归属

📋 测试步骤
1. 打开浏览器控制台 (F12)
2. 选择第一个项目 → 选择第一个 Session
3. 观察控制台日志
4. 点击下方按钮验证修复效果
🔍 验证标准
✅ 切换 Session 时看到: [SessionManager] Resetting state for session switch
✅ 两轮对话正确分开显示
✅ 工具调用通过 dialogId 正确匹配响应块
✅ 看到: [ChatManager] ✓ Found response block by dialogId
📝 控制台日志输出
